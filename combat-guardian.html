Combat Guardian: Under Attack

A strategic defense game where players protect their base from waves of enemies

Role: Unity Developer Platform: Mobile (Android/iOS) Genre: Strategy, Defense

Project Overview

Combat Guardian: Under Attack is an engaging mobile strategy game where players must defend their base against increasingly challenging waves of enemies. The game combines tower defense mechanics with real-time strategy elements, requiring players to think tactically about unit placement and resource management.

Key Features

  • Multiple tower types with unique abilities
  • Progressive difficulty system
  • Resource management mechanics
  • Upgrade system for towers and base
  • Multiple enemy types with different behaviors
  • Strategic placement mechanics

Technical Highlights

  • Optimized for mobile performance
  • Efficient pathfinding algorithms
  • Dynamic difficulty scaling
  • Smooth 60fps gameplay
  • Cross-platform compatibility

Development Process

Phase 1: Core Mechanics

Implemented basic tower defense mechanics including tower placement, enemy spawning, and pathfinding systems.

Phase 2: Gameplay Systems

Developed upgrade systems, resource management, and multiple tower types with unique abilities.

Phase 3: Polish & Optimization

Optimized performance for mobile devices, added visual effects, and fine-tuned game balance.

Challenges & Solutions

Performance Optimization

Challenge: Maintaining smooth 60fps gameplay with multiple enemies and effects on mobile devices.

Solution: Implemented object pooling for enemies and projectiles, optimized rendering with LOD systems, and used efficient algorithms for collision detection.

Balanced Gameplay

Challenge: Creating engaging difficulty progression that keeps players challenged without being frustrating.

Solution: Developed a dynamic difficulty system that adapts to player skill level and provides multiple difficulty modes for different player types.

Results & Impact

Player Engagement

Average session time of 15+ minutes with high retention rates

Performance

Consistent 60fps on mid-range mobile devices
